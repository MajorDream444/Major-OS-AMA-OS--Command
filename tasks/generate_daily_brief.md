# Task: Generate Daily Brief

## Objective

Generate the daily corporate brief for Major AI OS during the 30-day aggressive execution cycle.

## Operating Mode

Hybrid aggressive: build authority and monetize simultaneously.

## Required Outputs

Every daily brief must include:

1. Revenue Action
2. Content Action
3. Logged System Improvement
4. Blocker

## Procedure

1. Plan the highest-leverage daily actions.
2. Build or execute the smallest useful step.
3. Run the relevant local check or workflow.
4. Log the outcome in GitHub.
5. Improve the system based on the blocker or handoff.
